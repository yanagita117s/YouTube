Google Calendar API 設定

クライアントIDの取得手順：
1. Google Cloud Console を開く
2. プロジェクトを作成 → 「APIとサービス」→「ライブラリ」→ Google Calendar API を有効化
3. 「認証情報」→「+ 認証情報を作成」→「OAuth クライアントID」→「ウェブアプリケーション」
4. 承認済みのJavaScript生成元に null（ローカルファイル用）を追加、またはサーバーのURLを追加
5. 作成されたクライアントIDを下に貼り付けて保存
保存するとlocalStorageに記録されます（このデバイスのみ）

YouTube Data API 設定

YouTube用クライアントID（別アカウント用）：
1. Google Cloud ConsoleでYouTubeチャンネルのGoogleアカウントにログイン
2. 同じプロジェクトで「YouTube Data API v3」を有効化
3. OAuthクライアントIDを作成（または既存のものを使用）
4. OAuth同意画面の「データアクセス」に youtube.readonly スコープを追加
5. テストユーザーにYouTubeアカウントのメールを追加
空白のままにするとカレンダーと同じクライアントIDを使います
